## Commands

#### How to get started
```bash
git clone https://github.com/mju-likelion/livingodlife-backend.git
cd livingodlife-backend
npm install
npm run dev
```

#### Custom deploy
````bash
serverless deploy
# before deploy, you have to configure credentials
````
